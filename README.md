# 🍫 Chocolate Sales Analysis Dashboard

## 📘 Project Information

This project provides a comprehensive analysis of chocolate sales performance using Microsoft Power BI. The objective is to uncover sales trends, evaluate product performance, monitor profitability, and analyze regional and salesperson contributions through interactive dashboards and visualizations.

The project is built using multiple interconnected tables, providing a complete view of sales operations, profitability, customer demand, and business performance.

---

## 🎯 Business Problem

Business leaders need data-driven insights to improve sales performance, maximize profitability, and optimize product strategies. Key questions addressed in this project:

* Which products generate the highest sales and profit?
* Which regions contribute the most revenue?
* How are salespersons performing across different markets?
* What are the monthly and quarterly sales trends?
* Which products have the highest profit margins?
* Where are opportunities to improve profitability and sales growth?

This analysis delivers actionable insights through interactive Power BI dashboards to support strategic business decisions.

---

## 🗂️ Dataset

**Dataset Source:** Chocolate Sales Dataset

| # | Table Name | Description                                                           |
| - | ---------- | --------------------------------------------------------------------- |

## 🔁 Workflow

### Step 1: Data Cleaning

* Imported sales, product, salesperson, location, and calendar data into Power Query
* Removed duplicates and handled missing values
* Standardized date formats and validated data quality
* Created calculated columns for Year, Quarter, Month, Profit Margin, and Sales Metrics

### Step 2: Data Modeling

* Built relationships between Shipments, Products, People, Locations, and Calendar tables
* Implemented a star schema model for efficient reporting
* Created a dedicated Calendar table for time intelligence analysis

### Step 3: DAX Measures

Created key DAX measures including:

* Total Sales Revenue
* Total Boxes Sold
* Total Cost
* Gross Profit
* Profit Margin (%)
* Average Selling Price per Box
* Sales Growth (%)
* Region-wise Revenue Contribution

### Step 4: Dashboard Creation

Built multiple interactive dashboard pages including:

* Executive Dashboard
* Product-Level Analysis
* Geographic & Regional Analysis
* Salesperson Performance Analysis
* Time Intelligence & Trend Analysis
* Cost and Profit Strategy Dashboard

Used KPI cards, charts, maps, matrices, and slicers to create a dynamic business reporting experience.
